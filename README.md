# Training: Looker Studio

This repository contains materials for a hands-on training on Looker Studio (formerly Google Data Studio). It includes the presentation slides, assets, and example data used during the workshop.

- Presentation (live): http://surquest.github.io/training-lookerstudio
- Slides source: src/content/slides.md
- Assets and data: src/assets/ and src/data/

# Usage

Open the presentation link above to view the workshop slides. The repository contains the source files used to build and customize the slides.
